# Create an Animated Marp Presentation with Emojis

## Objective
Create an engaging Marp presentation about SonarQube with animations and emojis.

## Technical Requirements
1. Use Marp's built-in features for:
   - Slide transitions
   - CSS animations
   - Emoji support
   - Custom styling

## Structure
1. Create a new file with `.md` extension
2. Add the following frontmatter:

```yaml
---
marp: true
theme: default
paginate: true
style: |
  section {
    font-size: 1.5em;
  }
  h1 {
    font-size: 2em;
    color: #1a73e8;
  }
  h2 {
    font-size: 1.8em;
    color: #34a853;
  }
  h3 {
    font-size: 1.6em;
    color: #ea4335;
  }
  .emoji {
    font-size: 1.2em;
  }
  .fade {
    animation: fadeIn 1s ease-in;
  }
  @keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
  }
  .slide-up {
    animation: slideUp 0.5s ease-out;
  }
  @keyframes slideUp {
    from { transform: translateY(20px); opacity: 0; }
    to { transform: translateY(0); opacity: 1; }
  }
---
```
